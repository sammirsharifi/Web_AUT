Arrays

 

  • In JavaScript, arrays always use numbered indexes
    • Java Script does not support Arrays with named indexes (associative arrays) -In fact, Arrays are a special kind of objects, with numbered indexes.
      • If you use named indexes, JavaScript will redefine the array to an object.
    • Arrays with integer index (key)
      • var a = [1, 2, 3];(Array indexes start with 0)
      var a = new Array (10, 20, ...); (access to whole arry with the name(here "a"))
      var a = new Array();   a[10] = "HTML"; a[120] = "JS";
    • General associative array
      • var style = new Array(); var style = {color: "Blue", size: "20pt"};
      var c = style["color"]; var c = style.color;
    • In associative array, some array methods and properties will produce incorrect results(for example c.length)
