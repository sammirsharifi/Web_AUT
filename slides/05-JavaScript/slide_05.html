What is JavaScript?

 

  • A prototype-based, client-side, and scripting language to make web pages dynamic
  • Object-based: NOT class-based, no difference between classes & instances
  • Client-side: Run by client side program (the web browser)
  • Scripting: Doesn't require to be compiled before run( done on-the-fly by the client)
  • Additional facts about JS:
    • JavaScript is not related to Java.
    • Its standard name is ECMAScript
    • Sever side and other applications (e.g., GNOME Shell) are not discussed here.
