JavaScript Basic: Input & Output

 

  • To prompt a dialog to user and get input
    • var input = prompt("Please enter your name");
    • prompt returns string
      • To convert string to integer: var i = parseInt("10");
      • To convert string to float: var f = parseFloat("20.2");
  • To get confirmation from user
    • var question = confirm("Do you want to continue?");
    • confirm returns a boolean
      • Ok: True
      • Cancel: False
