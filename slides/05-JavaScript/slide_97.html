Debugging

 

  • Use the DOM tab to find methods & properties
  • Basic debugging via console, console.log()/info()/warn()/error()
  • Advanced debugging via real debuggers; e.g., FireFox debugger [1][2].
  • More secure coding with "use strict";
  • Best Practices
    • Avoid Global Variables
    • Always Declare Local Variables
    • Beware of Automatic Type Conversions
    • Use === Comparison
  • Use powerful IDEs for development
