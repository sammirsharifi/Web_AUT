Arrays

 

  • Methods: concat, shift, unshift, sort, reverse, ...
    • let friends = ["Ali", "Reza", "Hassan", "Kazem"];
    • friends.pop(); // Ali,Reza,Hassan
    • let newfriend = friends.concat("amir"); // Ali,Reza,Hassan,amir
    • newfriend.push("ahmad"); // Ali,Reza,Hassan,amir,ahmad
    • let x= newfriend.push("ahmad"); // 5: The push() method returns the new array length
    • let shifttest = newfriend.shift(); //Ali: removes the first array element
    • unshift():adds a new element at the beginning, and "unshifts"older elements:returns the new array length
    • let sorttest = friends.sort();//Ali,Hassan,Kazem,Reza : sorts an array alphabetically
    • reverse():reverses the elements in an array
  • for/of - looping over iterable objects : for (let friend of friends) {....;}
