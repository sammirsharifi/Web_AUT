Sample Libraries: Math, Number & Date Objects

 

  • Math:
    • abs, sin, ceil, floor, log, exp, pow, random, ...
    Examples: Math.abs(-11);//returns 11 Math.ceil(6.8);//returns 7 Math.random();//random number between 0 and 1
  • Number:
    • var num = 1.1
    num.toExponential and num.toFixed(converts a number to a string)
  • Date:
    • var d = new Date()
    d.toString():  
    d.set/get FullYear, Month, Date, Hours, Minutes, Seconds
