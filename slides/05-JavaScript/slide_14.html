JavaScript Basic: Operators

 

  • Arithmetic: + - * / % ++ --
    • String concatenation: +
  • Assignment: = += -= *= /= %=
  • Comparison: == === != !== > >= < <= (== , != check value regardless of operands type)
    • 2 == "2" returns true, 2 === "2" returns false
    • 2 != "2" returns false, 2 !== "2" returns true
  • Logical: && || !
  • Comments: // /* */
