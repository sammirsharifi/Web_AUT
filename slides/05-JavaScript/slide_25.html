Objects

 

  • How to access value of Objects:
    • Dot Notation : console.log(userdata.name) //'XXX' (or let userName = userdata.name)
    • Bracket Notation : console.log(userdata['name']) // "Ali"(or let userName = userdata['name'])
  • How to set value (like a simple value):userData.name = 'Amin'
  • Dot is a common way but: Consider you want to get key value from from input(prompt or input of html)
