DOM

 

  • Document Object Model (DOM) is the output of parsing XHTML file by browser
    • XHTML document is represented by DOM in browser
  • Each XHTML element is represented by an object
    • However, there are other objects (called BOM) that are not corresponding to HTML elements; e.g., window is the browser window
  • A tree of object corresponding to XHTML tag hierarchy
  • Each object has
    • Properties corresponding to XHMTL properties (not all properties)
    • Methods corresponsing to events and actions
